# 校园搭子小程序 (Campus Connect Mini Program)

## 🚀 项目简介 (Project Introduction)

"校园搭子"是一款专为在校大学生设计的社交小程序，旨在帮助同学们快速、精准地找到志同道合的伙伴，共同进行学习、运动、美食、娱乐等各类校园活动。项目致力于打破校园内的社交壁垒，让每个学生都能轻松发起和参与活动，丰富课余生活。

## ✨ 项目背景 (Background)

在现代大学生活中，尽管社交网络发达，但许多学生仍感到孤独，或在寻找有共同兴趣的伙伴参与线下活动时遇到困难。"想去打球却凑不齐人"、"想找人一起泡图书馆却不认识学霸"、"想拼单喝奶茶却找不到饭搭子"——这些都是校园生活的常见痛点。本项目应运而生，旨在通过一个轻量、便捷的平台，解决这些"找搭子"的难题。

## 🌟 主要功能 (Features)

- **智能匹配与发现**:
  - 首页采用**卡片式瀑布流**，无限下拉，高效发现潜在的"搭子"。
  - 支持按"推荐"、"最新"、"热门"等多种维度对用户卡片进行排序。
  - 集成**学校选择**和**关键词搜索**功能，实现精准查找。

- **完整的用户认证体系**:
  - 全新设计的**登录/注册页面**：支持手机号验证码登录、微信一键登录两种方式。
  - 表单实时校验（手机号正则、验证码长度、昵称要求），提供即时反馈。
  - 验证码倒计时功能（60秒），防止重复发送。
  - 用户协议勾选确认机制。

- **校园身份认证**:
  - **学生证认证**：上传学生证照片，3分钟快速审核。
  - **学号认证**：选择学校并输入学号信息进行验证。
  - 认证完成后引导用户设置兴趣标签，完善个人资料。
  - 认证状态全局同步，支持跨页面状态管理。

- **兴趣标签系统**:
  - "我的标签"和"期望搭子标签"双维度设置。
  - 标签分类涵盖学习、运动、游戏、场景、时间、性格等多个维度。
  - 至少选择3个标签才能完成设置，确保匹配质量。

- **活动广场**:
  - 用户可以发布或浏览**组队活动**，如"周末篮球局"、"图书馆学习小组"等。
  - 活动卡片清晰展示活动状态、时间、地点和已参与人数。

- **校园圈动态**:
  - 提供一个类似朋友圈的**信息流**，供用户分享图文动态，营造社区氛围。
  - 支持点赞、评论等基本社交互动。

- **即时通讯**:
  - 内置聊天功能，当用户匹配成功或对彼此感兴趣时，可**一键发起对话**，方便深入沟通。
  - 聊天功能采用**分包加载**，不影响主包性能。

- **丰富的个人中心**:
  - 未登录状态展示精美的登录/注册引导页，已登录状态显示完整用户资料。
  - 包含"我的收藏"、"积分商城"、"VIP特权"等模块。
  - 创新的**"宠物喂养"**游戏化功能，提升用户粘性和趣味性。
  - 实名认证进度提示卡，引导用户完善资料。
  - 紧急联系人设置，保障出行安全。

## 🛠️ 技术栈 (Tech Stack)

- **前端**: 原生微信小程序 (WXML, WXSS, JavaScript ES6+)
- **核心框架**: 无，采用小程序原生开发范式。
- **状态管理**: 小程序原生 `Page.data` + `App.globalData` 模式，支持跨页面状态同步。
- **UI 库**: 无，所有组件均为自定义实现，遵循 UI/UX Pro Max 设计规范。
- **代码风格**: 遵循 `ESLint` 规范。
- **组件按需注入**: 已启用微信小程序组件按需注入功能，优化加载性能。
- **分包策略**: 采用 sub-packages 分包加载，主包体积控制在限制范围内。

## 🎨 设计亮点 (Design Highlights)

### 视觉设计系统

1. **品牌紫色主题**:
   - 主色调采用紫色渐变系列（#7C3AED → #9333EA → #A855F7）。
   - 登录页面使用淡紫色渐变背景（#EDE9FE → #DDD6FE → #C4B5FD），柔和舒适。
   - 操作按钮使用紫色互补色——琥珀色/橙色（#F59E0B → #D97706），形成和谐对比。

2. **现代化登录/注册界面**:
   - 动态浮动光斑背景动画（blob animation），增加视觉层次感。
   - 毛玻璃品牌区（backdrop-filter blur），精致优雅。
   - 登录/注册 Tab 切换带底部指示器滑动动画。
   - 表单输入框聚焦时紫色边框 + 光晕反馈。
   - 微信一键登录按钮独立设计，琥珀色渐变突出 CTA。

3. **认证页面专业排版**:
   - 权益说明卡片使用虚线边框 + 渐变背景，视觉突出。
   - 认证方式卡片全宽适配屏幕，文字左对齐，布局整洁。
   - 提交按钮使用 Flexbox 居中方案，文字完美水平垂直居中。
   - 头部标题区域严格左对齐，保持一致性。

4. **响应式设计**:
   - 通过 `@media` 查询适配三种屏幕尺寸：
     - 超小屏（高度 < 600px）：压缩间距，隐藏非必要元素。
     - 小屏手机（高度 < 700px）：适度压缩内边距。
     - 大屏设备（高度 > 850px）：增加顶部间距，更舒展的布局。

### 产品设计亮点

5. **可解释性推荐算法**:
   - 本项目的核心亮点之一。在展示匹配度分数的同时，系统会用一句自然语言（如"你们都喜欢晚上去图书馆学习"）来解释推荐原因，让匹配不再是冷冰冰的数字，极大地提升了推荐的透明度和用户的信任感。

6. **高信息密度的卡片设计**:
   - 首页的用户卡片在有限空间内，通过精巧的布局融合了封面、匹配度、标签、信誉分、社交证明（Social Proof）等多种元素，帮助用户在快速浏览中高效获取关键信息，做出判断。

7. **手绘与 Emoji 风格的 UI**:
   - 整个小程序采用了一种轻松、活泼的手绘风格，并大量使用 Emoji 作为图标和装饰，营造出年轻、友好的社区氛围，精准命中目标用户群体的审美。

8. **游戏化体验设计**:
   - "宠物喂养"功能是提升用户留存的巧妙设计。它将用户的活跃行为（如登录、发布动态）转化为游戏内的激励，增加了产品的趣味性和长期吸引力。

## 🔐 用户流程 (User Flow)

```
打开小程序 → 首页浏览 → 点击"我的"(Profile)
    ↓
[未登录] → 登录/注册页面
    ├─ 手机号 + 验证码 登录
    ├─ 手机号 + 验证码 + 昵称 注册
    └─ 微信一键登录（快捷入口）
    ↓
登录成功 → 校园身份认证页面(Verification)
    ├─ 学生证认证（上传照片）
    └─ 学号认证（学校 + 学号）
    ↓
认证完成 → 兴趣标签设置页面(Tags) [可选]
    ↓
返回个人中心(Profile) → 完整功能解锁
```
